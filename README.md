# Flow Builder

WeWeb custom component for building automation flows.

## Installation

```bash
npm install
```

## Development

```bash
npm run serve
```

The component will be available at `https://localhost:8080/`

## Usage

Add to WeWeb:
1. Open your WeWeb project
2. Go to Custom Elements Developer popup
3. Add the URL: `https://localhost:8080/manager.js`
